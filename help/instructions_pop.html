Adding Two-Factor Authentication to your account

Two factor authentication is a way to secure your account, even in the event that a malicious party steals your password.

1. Get the Software


2. Set it up

There is currently a known issue with iPhones that prevents them from scanning barcodes. Please be sure to read about manually inputting your key if you use an iPhone

On the two-factor authentication page, once you've logged in, scan the barcode shown. If you can't, click the link to manually input your key. Type in the key shown, ignoring spaces.

You device should now show a six-digit code that changes.

3. Use It

Input the code you see to verify that it's working correctly, and you're done!
